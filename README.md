# Active AI System

Complete outreach, automation, and sales management system for Active AI agency.

## Features

- **Outreach Manager**: Track prospects, manage conversations, monitor engagement
- **Outreach Automation**: Schedule messages, manage daily targets, track response rates
- **Sales System**: Discovery call script, proposal generator, pricing framework

## Quick Deploy to Vercel

### Step 1: Prepare Your Project
1. Go to GitHub.com and create a new repository called `active-ai-system`
2. Clone the repository to your computer

### Step 2: Add Project Files
1. Copy all files from this folder into your GitHub repository
2. Push to GitHub:
```bash
git add .
git commit -m "Initial commit"
git push origin main
```

### Step 3: Deploy to Vercel
1. Go to vercel.com and sign up (free)
2. Click "New Project"
3. Select your GitHub repository
4. Click "Deploy"
5. Vercel will automatically build and deploy your app
6. You'll get a live URL in seconds

## Local Development (Alternative)

If you want to run this locally:

```bash
npm install
npm start
```

The app will open at http://localhost:3000

## How to Use

### Outreach Manager
1. Import your leads list (the Excel file)
2. Search for prospects
3. Select each one and compose personalized messages
4. Track who you've messaged and who's replied

### Outreach Automation
1. Set your daily message target (5-50)
2. Configure time slots for sending
3. Select focus area (Customer Service, CRM, etc.)
4. Click "Start Campaign"
5. Monitor response rate and estimated conversions

### Sales System
1. **Discovery Tab**: Review the 10-question call script before taking calls
2. **Proposal Tab**: Fill in company details, pain points, automation area
3. **Pricing Tab**: Reference your pricing framework during sales conversations
